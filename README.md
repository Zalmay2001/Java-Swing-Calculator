# 🧮 Java Swing Calculator

A clean, functional desktop calculator built with **Java Swing**. This project demonstrates fundamental GUI programming, event-driven design, and the MVC pattern in a classic desktop environment.

![Calculator Screenshot](screenshot.png)  
*(Take a screenshot of your running app, save it as `screenshot.png` in the root folder, and it will appear here!)*

## ✨ Features
- ➕ Addition, Subtraction, Multiplication, Division
- ⌫ Backspace and Clear (C) functionality
- 🔢 Decimal point support
- ⌨️ Keyboard input support *(if you implemented it, keep this; otherwise remove)*
- Responsive and intuitive layout

## 🛠️ Tech Stack
- **Java** (JDK 8+)
- **Swing** (GUI Library)
- **NetBeans** IDE (optional, but supported)

## 🚀 How to Run

### Option 1: Using NetBeans (Easiest)
1. Open the project folder in NetBeans.
2. Right-click the project and select **Run**.

### Option 2: Compile & Run via Terminal
Navigate to the `src` directory and compile:
```bash
cd src
